# Opera

Opera是Opera公司开发的一款浏览器，从1995年发布更新到至今。

# Opera的内核 Presto

从1995年到2003年之前不清楚内核是啥，2003年到2013年使用的是Presto，2013年以后Opera是基于Chromium重新开发的，内核是BLink和V8.
